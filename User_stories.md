 User Story Backlog (Initial)
| User Story | Role | Goal | Priority | Estimate (days) |
|------------|------|------|---------|----------------|
| Browse restaurants | Customer | As a customer, I want to browse available restaurants so I can choose where to order | High | 2 |
| View menu | Customer | As a customer, I want to see the menu for a restaurant so I can select my meal | High | 2 |
| Place order | Customer | As a customer, I want to place an order and pay online so that I can receive food conveniently | High | 3 |
| Track order | Customer | As a customer, I want to see my order status in real-time so I know when it will arrive | Medium | 2 |
| Rate restaurant | Customer | As a customer, I want to leave a rating and review so that other users can benefit | Low | 1 |
| Save favorite restaurants | Customer | As a customer, I want to save restaurants to a favorites list so I can reorder easily | Medium | 1 |
| Apply promo codes | Customer | As a customer, I want to apply discount codes to my orders so I can save money | Low | 1 |
| Filter meals by dietary preference | Customer | As a customer, I want to filter menu items by vegan, keto, or gluten-free so I can eat according to my diet | Medium | 2 |
| Subscription / loyalty program | Customer | As a customer, I want to join a loyalty program so I can get discounts for frequent orders | Low | 2 |
| Manage account/profile | Customer | As a customer, I want to create and manage my account, update personal info and payment methods so I can securely use the app | High | 2 |
| Manage menu | Restaurant | As a restaurant, I want to add, edit, or remove menu items so I can update offerings | High | 2 |
| Receive orders | Restaurant | As a restaurant, I want to receive notifications for new orders so I can prepare them on time | High | 2 |
| Respond to messages | Restaurant | As a restaurant, I want to reply to customer messages so I can clarify orders | Medium | 1 |
| View analytics | Restaurant | As a restaurant, I want to see orders, revenue, and popular items so I can make business decisions | Low | 2 |
| Manage availability | Restaurant | As a restaurant, I want to set open/close times and delivery options so customers see accurate info | Medium | 1 |
| Highlight featured dishes | Restaurant | As a restaurant, I want to feature certain menu items so I can boost sales | Low | 1 |
| Receive customer feedback | Restaurant | As a restaurant, I want to view ratings and reviews so I can improve service | Medium | 1 |
| View payout before accepting delivery | Driver | As a part-time driver, I want to see the payout before accepting a delivery so I can decide if the order is worth my time | High | 1 |
| Receive clear pickup/drop-off instructions | Driver | As a part-time driver, I want to receive clear pickup and drop-off instructions so I can complete deliveries efficiently | High | 1 |
| Fast order assignments | Driver | As a part-time driver, I want fast order assignments with minimal idle time so I can maximize earnings | Medium | 1 |
| Track earnings and incentives | Driver | As a part-time driver, I want to track total earnings and incentives in the app so I know my earnings | Medium | 1 |
| In-app support for delivery issues | Driver | As a part-time driver, I want in-app support for delivery issues so I can resolve problems quickly | Medium | 1 |
| Stable order flow | Driver | As a full-time driver, I want a stable flow of delivery requests so I can maintain consistent daily income | High | 2 |
| Accurate distance & payout estimates | Driver | As a full-time driver, I want accurate distance and payout estimates before accepting orders so I can plan my schedule | High | 1 |
| View performance tracking and incentives | Driver | As a full-time driver, I want to view performance tracking and incentives so I can improve earnings and maintain a strong rating | Medium | 1 |
| Safety features & emergency support | Driver | As a full-time driver, I want safety features and emergency support so I can stay safe while working | High | 1 |
| Communication channels with customers & restaurants | Driver | As a full-time driver, I want clear communication channels with customers and restaurants so I can avoid delays and misunderstandings | Medium | 1 |
| Admin dashboard | Admin | As an admin, I want to monitor restaurants and orders so I can ensure platform quality | Medium | 2 |
| Monitor platform activity | Admin | As an admin, I want to monitor all active orders and platform activity in a centralized dashboard so I can ensure reliability | High | 2 |
| Handle refunds, complaints, disputes | Admin | As an admin, I want to handle refunds, complaints, and disputes so I can resolve user issues efficiently | High | 2 |
| Manage user and restaurant accounts | Admin | As an admin, I want to manage user and restaurant accounts so I can suspend fraudulent accounts or fix problems | Medium | 2 |
| Real-time delivery issue & performance tracking | Admin | As an admin, I want to see delivery issues and performance metrics in real-time so I can identify bottlenecks and improve service | Medium | 2 |
| Generate analytics & reports | Admin | As an admin, I want to generate analytics and reports so I can make data-driven decisions for platform growth | Medium | 2 |

> Note: Additional user stories will be added as the project progresses.
