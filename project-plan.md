# FeedMe Project Plan (Step 1(Planning))

## 1. Market Research

### Competitor Analysis
| Competitor   | Strengths                                   | Weaknesses / Gaps                          |
|-------------|--------------------------------------------|--------------------------------------------|
| FoodPanda   | Large selection, fast delivery, easy app   | High delivery fees, small/local vendors less visible |
| UberEats    | User-friendly, reliable tracking           | Expensive for some users, limited healthy/affordable filters |
| Deliveroo   | Good restaurant variety                     | Limited options for small restaurants      |
| Menulog     | Simple interface, multiple payment options | UX less modern, few loyalty features       |

**Opportunity / USP for FeedMe:**
- Highlight small/local restaurants  
- Affordable delivery pricing  
- Healthy and specialty filters (e.g., vegan, keto)  
- Optional subscription / loyalty program  

---

## 2. User Personas

### Customers (End-Users)
- **Students:** Budget-conscious, busy, need fast ordering  
- **Families:** Want healthy options, multiple meals at once  
- **Professionals:** Time-poor, value speed and convenience  

### Restaurants (Service Providers)
- **Small cafes / food trucks:** Need more visibility  
- **Medium local restaurants:** Want simple order management  
- **Focus:** Support small/local businesses  

---

## 3. Core Features

### For Customers
- Browse restaurants & menus  
- Place orders & pay online securely  
- Track order in real-time  
- Leave ratings & reviews  
- Manage profile & account  

### For Restaurants
- Manage menu items & pricing  
- Receive and manage incoming orders  
- View basic analytics (orders, revenue)  
- Respond to customer messages  
- Manage availability / delivery options  

---

## 4. Unique Features (USP)
- Highlight small/local restaurants on the platform  
- Affordable / transparent delivery pricing  
- Filter meals by dietary preference or affordability  
- Optional subscription or loyalty program  
- Future iteration: AI-powered personalized recommendations  

---

## 5. Technologies (Planned)
- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Database:** MySQL  
- **Version Control:** Git & GitHub  
- **Cloud Services:** AWS (planned for later iterations)  

---

## 6. Project Management
- **Methodology:** Agile (Scrum-style)  
- **Tools:**  
  - **GitHub Projects** – for version control and issue tracking  
  - **Figma** – for wireframes and UI/UX prototyping  
  - **Trello** – for task and iteration management  
- **Wireframing / Prototyping:** Basic layout sketches for home page, menu page, checkout  
- **UX/UI:** Ensure easy navigation, responsive design, accessibility basics  
- **Security:** Passwords stored securely, sensitive data encrypted, role-based access  
- **Testing (later iterations):** Backend unit tests, frontend usability tests  
- **Presentation:** Plan demo: browse restaurants, place orders, view order history
